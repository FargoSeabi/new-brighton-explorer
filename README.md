# New Brighton Explorer

A modern web application designed to help users explore and discover information, locations, activities, and experiences in New Brighton. This project provides an intuitive and user-friendly interface for browsing content and interacting with local attractions and services.

---

## Features

- 🌍 Explore destinations and attractions in New Brighton
- 📍 Interactive and user-friendly navigation
- 🔎 Search and filter functionality
- 📱 Responsive design for desktop and mobile devices
- ⚡ Fast and lightweight application performance
- 🎨 Clean and modern UI

## Installation

Clone the repository:

```bash
git clone https://github.com/FargoSeabi/new-brighton-explorer.git
```

Navigate to the project folder:

```bash
cd new-brighton-explorer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Usage

1. Open the application in your browser.
2. Browse attractions and available information.
3. Use the search feature to find specific places or activities.
4. Explore the interactive interface and discover New Brighton.
